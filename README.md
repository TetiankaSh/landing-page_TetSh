# Bang & Olufsen Landing Page

A premium, responsive landing page concept for Bang & Olufsen, highlighting high-end audio products through a minimalist and sophisticated design. The project focuses on semantic HTML structure, modular SCSS architecture, and interactive user interface elements.

## Design Reference
This project was implemented based on the following professional design:
* **[Figma Design Link](https://www.figma.com/design/DtkQmQ797hk0nI4KfMi2Uq/BOSE-New-Version?node-id=6817-212)**

## Technologies Used
* **HTML5** – Semantic markup for better SEO and accessibility.
* **SCSS** – Modular architecture with 15+ files, utilizing variables, mixins, and nested rules.
* **BEM Methodology** – For clear, maintainable, and scalable CSS class naming.
* **JavaScript** – For handling mobile menu toggles and smooth scroll interactions.

## Getting Started
Follow these steps to set up the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/TetiankaSh/landing-page_TetSh.git
cd landing-page_TetSh
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the project locally

```bash
npm start
```
